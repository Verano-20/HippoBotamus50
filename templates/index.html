This is HippoBotamus50 - reborn!

By Sam Dawson, Ely, UK

GitHub: Verano-20


This is a 'reboot' of my CS50x final project which I completed back in April 2020. The reboot's main purpose is to add Javascript features to show off some more front-end work.

Powered by Flask, Bootstrap, Javascript and InstaPy, this website is linked to the Instagram account @hippobotamus50.

Using this account, there are several functions built into this site enabling you to interact with HippoBotamus50 automatically.

I chose this as my project as an exploration into the automation of social media accounts. Using open-source resources in python, it has been relatively straightforward to create a python script to automate following and liking posts on Instagram. In this age of misinformation and manipulation online, and where social media is a powerful tool for influencing public opinion, I wanted to see just how easy it was to make a bot account.
